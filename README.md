## Correrlo de manera local

Clonar el proyecto

```bash
  git clone https://github.com/JairoBoss/store4me
```

Ir al directorio del proyecto

```bash
  cd store4me
```

### api

Instalar dependencias

```bash
  cd api    
  npm install
```

Configurar variables de entorno
 
 * PORT
 * MONGO_PORT
 * MONGO_DB
 * TOKEN_SECRET
 * AWS_BUCKET_NAME
 * AWS_REGION
 * AWS_ACCESS_KEY
 * AWS_SECRET_KEY
 * PAYPAL_API_CLIENT
 * PAYPAL_API_SECRECT

Correr la api

```bash
  npm start
```

### front

Instalar dependencias

```bash
  cd front
  npm install
```

Correr el front

```bash
  npm start
```
